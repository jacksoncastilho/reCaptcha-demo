# reCaptcha-demo

Make sure you have `php-curl` installed in you server
```
php -m | grep curlv
```
If "curl" is listed, cURL is enabled.

If not, you need to install it. Open your terminal and run
```
sudo apt-get update && sudo apt-get install php-curl
```
